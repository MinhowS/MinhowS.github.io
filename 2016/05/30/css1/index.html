css基础2

发表于 2016-05-30   |   分类于 前端   |     |   阅读次数

这里简单介绍下选择器,选择器也是重要的一部分了.先介绍下基础选择器.

一、基础选择器

1 标签选择器

结构如下：

1        <style type="text/css">
2            span{
3                color:red;
4            }
5        </style>

效果：会将所有的这类标签都加上相同的属性。

2 类选择器

.就是类的符号。类的英语叫做class。
所谓的类，就是class属性，任何的标签都可以携带class属性。class属性可以重复。
例如： <p class=”bq”>哈哈哈</p>

结构：

<style>
.类名（选择器） {
属性名1：属性值1;
属性名2：属性值2;
.......
}
</style>

效果：会将所有拥有这个类名的那些标签都加上相同的属性。

3 id选择器

与类选择基本上相同，id选择器的选择符是“#”。
任何的HTML标签都可以有id属性。表示这个标签的名字。
一个HTML页面，不能出现相同的id，哪怕他们不是一个类型。比如页面上有一个id为pp的p，一个id为pp的div，是非法的！

4 类选择器和id选择器的命名规则

1） 只能有字母、数字、下划线（0-9，a-z，A-Z，_,-）
2） 必须以字母开头
3） 不能和标签同名。比如id不能叫做body、img、a

5 id选择器与类选择器之间区别

类名：相当于是人的姓名。（可以重复）
特点：
1.0一个类名可以设置给多个标签
2.0一个标签可以有多个类名
id名：相当于是身份证号码。（必须唯一）
特点：
1.0只能给一个标签设置这个id名
2.0一个标签只能有一个id名。
id不是给css使用的，而是javascript来使用的。

二、高级选择器

1 通配符*

*就表示所有元素。 

1    *{
2        color:red;3    }

会去页面上一个一个的遍历页面的标签，然后给他们进行设置。效率不高，如果页面上的标签越多，效率越低，所以页面上不能出现这个选择器。

2 并集选择器（分组选择器）

1    h3,li{
2        color:red;3    }

用逗号就表示并集。

3 交集选择器

1    h3.special{
2        color:red;3    }

选择的元素是同时满足两个条件：必须是h3标签，然后必须是special标签。
交集选择器没有空格。
所以

1    div.red

和

1    div .red

不是一个意思。
交集选择器可以连续交（一般不要这么写）

1    h3.special.zhongyao{
2        color:red;
3    }

交集选择器，我们一般都是以标签名开头，比如div.haha 比如p.special。

4 后代选择器

1        <style type="text/css">
2            .div1 p{
3                color:red;
4            }
5        </style>

空格就表示后代，.div1 p 就是.div1的后代所有的p。
后代选择器，描述的是祖先结构。

5 子代选择器

作用：找到一个标签的直接子元素来设置样式。
